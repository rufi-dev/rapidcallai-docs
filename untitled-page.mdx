If you have worked through the relevant troubleshooting section and the issue is not resolved, reach out to us directly. We are fastest on live chat and can usually diagnose platform issues within minutes.

When contacting support, include the following so we can help you quickly:

**For call issues: **the call ID or recording URL from Call History, the agent name, the phone number involved, and a brief description of what went wrong.

**For billing issues: **the invoice date and amount, the specific line item in question, and the date range of the calls you are querying.

**For agent behaviour issues: **the call ID (so we can read the transcript), the step in the script where the agent went wrong, and what you expected to happen versus what actually happened.
