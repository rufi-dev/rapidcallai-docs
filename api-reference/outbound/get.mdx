---
title: "Get Outbound Job"
api: GET https://api.rapidcall.ai/api/outbound/jobs/{id}
description: "Get detailed status and progress for a specific outbound job."
---

Returns full details for an outbound job, including per-contact status breakdown and execution progress.

<ParamField path="id" type="string" required>
  The unique identifier of the outbound job.
</ParamField>

<RequestExample>
```bash cURL
curl -X GET https://api.rapidcall.ai/api/outbound/jobs/job_4a5b6c7d \
  -H "Authorization: Bearer YOUR_API_KEY"
```
</RequestExample>
