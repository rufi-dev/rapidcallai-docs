---
title: "Get Job Call Logs"
api: GET https://api.rapidcall.ai/api/outbound/jobs/{id}/logs
description: "Get individual call logs and outcomes for an outbound job."
---

Returns the per-contact call logs for an outbound job. Each entry includes the contact's phone number, call outcome, duration, and the associated call ID for further inspection.

<ParamField path="id" type="string" required>
  The unique identifier of the outbound job.
</ParamField>

<RequestExample>
```bash cURL
curl -X GET https://api.rapidcall.ai/api/outbound/jobs/job_4a5b6c7d/logs \
  -H "Authorization: Bearer YOUR_API_KEY"
```
</RequestExample>
