---
title: "List Outbound Jobs"
api: GET https://api.rapidcall.ai/api/outbound/jobs
description: "List all outbound calling jobs in your workspace."
---

Returns a list of all outbound jobs for the current workspace, ordered by creation date (newest first). Includes status, progress counters, and settings for each job.

<RequestExample>
```bash cURL
curl -X GET https://api.rapidcall.ai/api/outbound/jobs \
  -H "Authorization: Bearer YOUR_API_KEY"
```
</RequestExample>
