---
title: "Pricing Rates"
api: GET https://api.rapidcall.ai/api/billing/pricing
description: "Get current pricing rates for all supported models, voice providers, and telephony."
---

Returns the current pricing schedule for all billable resources. Rates are dynamically sourced and reflect the latest pricing for each LLM model, TTS/STT provider, and telephony usage. All prices are in USD.

RapidCall AI uses a pay-as-you-go model. You are charged based on actual consumption with no minimum commitments.

<RequestExample>
```bash cURL
curl -X GET https://api.rapidcall.ai/api/billing/pricing \
  -H "Authorization: Bearer YOUR_API_KEY"
```
</RequestExample>
