---
title: "List Invoices"
api: GET https://api.rapidcall.ai/api/billing/invoices
description: "List all invoices for your workspace with status, amount, and payment details."
---

Returns a list of invoices for your workspace. Invoices are generated monthly by Lago and synced to Stripe for payment collection.

<RequestExample>
```bash cURL
curl -X GET https://api.rapidcall.ai/api/billing/invoices \
  -H "Authorization: Bearer YOUR_API_KEY"
```
</RequestExample>
