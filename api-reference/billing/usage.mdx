---
title: "Recent Usage"
api: GET https://api.rapidcall.ai/api/billing/recent-usage
description: "Get per-call cost breakdown for recent calls in the current billing period."
---

Returns a list of recent calls with their individual cost breakdowns. Each entry shows the LLM, TTS, STT, and telephony costs for a single call. Useful for auditing charges and understanding per-call economics.

<RequestExample>
```bash cURL
curl -X GET https://api.rapidcall.ai/api/billing/recent-usage \
  -H "Authorization: Bearer YOUR_API_KEY"
```
</RequestExample>
