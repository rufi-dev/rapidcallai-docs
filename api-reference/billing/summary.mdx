---
title: "Billing Summary"
api: GET https://api.rapidcall.ai/api/billing/summary
description: "Get current billing period summary including month-to-date total, ongoing charges, and phone number costs."
---

Returns a summary of the current billing period for your workspace. Includes the running total for the month, any in-progress call charges, amounts already paid, and phone number subscription costs.

<RequestExample>
```bash cURL
curl -X GET https://api.rapidcall.ai/api/billing/summary \
  -H "Authorization: Bearer YOUR_API_KEY"
```
</RequestExample>
