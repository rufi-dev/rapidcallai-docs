---
title: Get Workflow
api: GET https://api.rapidcall.ai/api/agents/{id}/workflow
description: "Retrieve the current workflow graph for an agent"
---

Retrieve the current workflow graph for an agent, including both the draft and published versions.

<ParamField path="id" type="string" required>
  The agent ID.
</ParamField>

<RequestExample>
```bash cURL
curl https://api.rapidcall.ai/api/agents/g4d3cXIpDr/workflow \
  -H "Authorization: Bearer YOUR_TOKEN"
```
</RequestExample>

<ResponseExample>
```json 200
{
  "draft": {
    "nodes": [
      {
        "id": "node_1",
        "type": "greeting",
        "data": { "message": "Hello! How can I help you today?" },
        "position": { "x": 100, "y": 50 }
      },
      {
        "id": "node_2",
        "type": "question",
        "data": { "question": "Are you calling about an existing order or a new inquiry?" },
        "position": { "x": 100, "y": 200 }
      }
    ],
    "edges": [
      { "id": "edge_1", "source": "node_1", "target": "node_2" }
    ],
    "updatedAt": "2026-04-01T10:00:00Z"
  },
  "published": null
}
```
</ResponseExample>
