---
title: Reset Password
api: POST https://api.rapidcall.ai/api/auth/reset-password
description: "Reset password using a reset token"
---

Reset the account password using the token from the password reset email.

<ParamField body="token" type="string" required>
  Reset token from the email link.
</ParamField>

<ParamField body="password" type="string" required>
  New password (min 8 characters).
</ParamField>

<RequestExample>
```bash cURL
curl -X POST https://api.rapidcall.ai/api/auth/reset-password \
  -H "Content-Type: application/json" \
  -d '{
    "token": "reset_token_from_email",
    "password": "newSecurePass456"
  }'
```
</RequestExample>

<ResponseExample>
```json 200
{
  "message": "Password reset successfully.",
  "token": "eyJhbGciOiJIUzI1NiIs..."
}
```
</ResponseExample>
