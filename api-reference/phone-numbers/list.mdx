---
title: List Phone Numbers
api: GET /api/phone-numbers
description: "Get all phone numbers in the current workspace"
---

## Request

```bash
curl -H "Authorization: Bearer YOUR_TOKEN" \
  https://api.rapidcall.ai/api/phone-numbers
```

## Response

```json
{
  "phoneNumbers": [
    {
      "id": "pn_abc123",
      "workspaceId": "ws_xyz",
      "e164": "+15551234567",
      "label": "Main Office",
      "provider": "twilio",
      "status": "active",
      "twilioNumberSid": "PN...",
      "livekitInboundTrunkId": "ST_...",
      "livekitOutboundTrunkId": "ST_...",
      "livekitSipUsername": null,
      "livekitSipPassword": null,
      "sipTerminationUri": null,
      "sipOutboundTransport": null,
      "inboundAgentId": "agent_xyz",
      "outboundAgentId": null,
      "allowedInboundCountries": ["all"],
      "allowedOutboundCountries": ["all"],
      "source": "purchased",
      "createdAt": "2025-01-10T09:00:00Z",
      "updatedAt": "2025-01-10T09:00:00Z"
    }
  ]
}
```

<Info>
- The response wraps numbers in a `phoneNumbers` array.
- `provider` is `"twilio"` for purchased numbers or `"sip"` for connected SIP trunk numbers.
- `source` is `"purchased"` (billed monthly) or `"connected"` (BYO number).
- `inboundAgentId` and `outboundAgentId` indicate which agents handle calls on this number.
</Info>
