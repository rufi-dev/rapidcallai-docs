<ParamField body="e164" type="string" required>
  Phone number in E.164 format (e.g., `"+14151234567"`).
</ParamField>

<ParamField body="label" type="string">
  Display name for the phone number.
</ParamField>

<ParamField body="provider" type="string">
  `"twilio"` or `"sip"` (auto-detected from fields if not specified).
</ParamField>

<ParamField body="sipTerminationUri" type="string">
  SIP trunk termination URI. Required for SIP provider numbers.
</ParamField>

<ParamField body="sipUsername" type="string">
  SIP authentication username.
</ParamField>

<ParamField body="sipPassword" type="string">
  SIP authentication password.
</ParamField>

<ParamField body="sipOutboundTransport" type="string">
  SIP transport protocol: `"tcp"`, `"udp"`, or `"tls"` (default: `"tls"`).
</ParamField>

## Search Available Numbers

First, search for available Twilio numbers:

```bash
curl -H "Authorization: Bearer YOUR_TOKEN" \
  "https://api.rapidcall.ai/api/workspaces/ws_123/twilio/available-numbers?country=US&area_code=415"
```

### Response

```json
{
  "numbers": [
    {
      "phoneNumber": "+14151234567",
      "friendlyName": "(415) 123-4567",
      "locality": "San Francisco",
      "region": "CA",
      "isoCountry": "US",
      "capabilities": { "voice": true, "SMS": true, "MMS": true }
    }
  ]
}
```

## Buy a Number (Twilio)

Numbers are purchased through the workspace Twilio endpoint:

```bash
curl -X POST https://api.rapidcall.ai/api/workspaces/ws_123/twilio/buy-number \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "phoneNumber": "+14151234567"
  }'
```

The number is automatically provisioned with inbound webhooks, SIP trunk, and status callbacks.

## Connect Existing SIP Number

Add a number you already own via SIP:

```bash
curl -X POST https://api.rapidcall.ai/api/phone-numbers \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "e164": "+14151234567",
    "label": "Office Line",
    "provider": "sip",
    "sipTerminationUri": "sip:trunk.your-provider.com",
    "sipUsername": "user123",
    "sipPassword": "pass456",
    "sipOutboundTransport": "tls"
  }'
```

### Response

```json
{
  "phoneNumber": {
    "id": "pn_new123",
    "workspaceId": "ws_xyz",
    "e164": "+14151234567",
    "label": "Office Line",
    "provider": "sip",
    "status": "unconfigured",
    "inboundAgentId": null,
    "outboundAgentId": null,
    "source": "connected",
    "createdAt": "2025-01-15T14:00:00Z",
    "updatedAt": "2025-01-15T14:00:00Z"
  }
}
```

<Warning>
A payment method is required before purchasing or connecting phone numbers.
</Warning>
