---
title: Update Phone Number
api: PUT /api/phone-numbers/{id}
description: "Update a phone number's agent assignment and settings"
---

## Request

```bash
curl -X PUT https://api.rapidcall.ai/api/phone-numbers/pn_abc123 \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "inboundAgentId": "agent_xyz",
    "label": "Sales Line"
  }'
```

## Parameters

| Field | Type | Description |
|-------|------|-------------|
| `label` | string | Display name |
| `inboundAgentId` | string/null | Agent to handle inbound calls (null to unassign) |
| `outboundAgentId` | string/null | Agent to handle outbound calls (null to unassign) |
| `sipTerminationUri` | string/null | SIP trunk termination URI |
| `livekitSipUsername` | string/null | SIP auth username |
| `livekitSipPassword` | string/null | SIP auth password |
| `sipOutboundTransport` | string/null | `"tcp"`, `"udp"`, or `"tls"` |
| `allowedInboundCountries` | string[] | Countries allowed for inbound calls |
| `allowedOutboundCountries` | string[] | Countries allowed for outbound calls |

All fields are optional — only provided fields are updated.

```json
    "updatedAt": "2025-01-15T15:00:00Z"
  }
}
```
