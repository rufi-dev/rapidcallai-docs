---
title: "Agent Usage Summary"
api: GET https://api.rapidcall.ai/api/agents/{id}/usage-summary
description: "Get token and voice usage summary for a specific agent."
---

Returns a breakdown of resource consumption for an agent, including LLM token usage, TTS minutes, and STT minutes. Useful for understanding cost drivers and optimizing agent configurations.

<ParamField path="id" type="string" required>
  The unique identifier of the agent.
</ParamField>

<ParamField query="startDate" type="string">
  Start of the date range in ISO 8601 format. Defaults to current billing period start.
</ParamField>

<ParamField query="endDate" type="string">
  End of the date range in ISO 8601 format. Defaults to now.
</ParamField>

<RequestExample>
```bash cURL
curl -X GET "https://api.rapidcall.ai/api/agents/agent_9k2m4n6p/usage-summary?startDate=2026-03-01T00:00:00Z" \
  -H "Authorization: Bearer YOUR_API_KEY"
```
</RequestExample>
