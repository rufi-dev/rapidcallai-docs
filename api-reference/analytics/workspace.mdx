---
title: "Workspace Analytics"
api: GET https://api.rapidcall.ai/api/analytics
description: "Get workspace-level analytics including total calls, duration, latency, and outcome breakdowns."
---

Returns aggregated analytics for your entire workspace over a given date range. Includes call volume, total duration, average latency, and a breakdown of call outcomes.

<ParamField query="startDate" type="string">
  Start of the date range in ISO 8601 format. Defaults to 30 days ago.
</ParamField>

<ParamField query="endDate" type="string">
  End of the date range in ISO 8601 format. Defaults to now.
</ParamField>

<RequestExample>
```bash cURL
curl -X GET "https://api.rapidcall.ai/api/analytics?startDate=2026-03-01T00:00:00Z&endDate=2026-03-31T23:59:59Z" \
  -H "Authorization: Bearer YOUR_API_KEY"
```
</RequestExample>
