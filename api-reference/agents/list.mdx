---
title: List Agents
api: GET https://api.rapidcall.ai/api/agents
description: "Get all agents in the current workspace"
---

<ParamField header="Authorization" type="string" required>
  Bearer token or API key. Example: `Bearer rck_your_api_key`
</ParamField>

## Request

```bash
curl -H "Authorization: Bearer YOUR_TOKEN" \
  https://api.rapidcall.ai/api/agents
```
