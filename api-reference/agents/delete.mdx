---
title: Delete Agent
api: DELETE https://api.rapidcall.ai/api/agents/{id}
description: "Delete an agent"
---

<ParamField header="Authorization" type="string" required>
  Bearer token or API key. Example: `Bearer rck_your_api_key`
</ParamField>

<ParamField path="id" type="string" required>
  The agent ID. Example: `agent_abc123`
</ParamField>

## Request

```bash
curl -X DELETE https://api.rapidcall.ai/api/agents/agent_abc123 \
  -H "Authorization: Bearer YOUR_TOKEN"
```

## Response

```json
{
  "ok": true
}
```

<Warning>
Deleting an agent also removes its phone number assignment. Any active calls will continue but no new calls will be routed.
</Warning>
