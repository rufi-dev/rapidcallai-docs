---
title: Create Agent
api: POST /api/agents
description: "Create a new voice agent"
---

## Request

```bash
curl -X POST https://api.rapidcall.ai/api/agents \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Sales Bot",
    "promptDraft": "You are a sales qualification agent...",
    "welcome": {
      "mode": "agent",
      "aiMessageMode": "static",
      "aiMessageText": "Hi! Thanks for calling. How can I help you today?"
    },
    "voice": {
      "provider": "cartesia",
      "model": "sonic-2",
      "voiceId": "a0e99841-438c-4a64-b679-ae501e7d6091"
    },
    "llmModel": "gpt-4o",
    "language": "en",
    "enabledTools": ["end_call", "transfer_call"],
    "agentType": "inbound",
    "role": "reception"
  }'
```

## Parameters

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `name` | string | Yes | Agent display name (max 60 chars) |
| `promptDraft` | string | — | Draft system prompt |
| `promptPublished` | string | — | Published (live) system prompt |
| `welcome` | object | — | Welcome message configuration |
| `welcome.mode` | string | — | `"user"` (wait for user) or `"agent"` (agent speaks first) |
| `welcome.aiMessageMode` | string | — | `"static"` (fixed text) or `"dynamic"` (LLM-generated) |
| `welcome.aiMessageText` | string | — | Static welcome text (when mode is static) |
| `welcome.aiDelaySeconds` | number | — | Delay before agent speaks |
| `voice` | object | — | TTS voice configuration |
| `voice.provider` | string | — | `"cartesia"`, `"elevenlabs"`, `"nvidia"` |
| `voice.model` | string | — | Provider model ID |
| `voice.voiceId` | string | — | Voice ID from the provider |
| `llmModel` | string | — | LLM model (e.g., `gpt-4o`, `claude-sonnet-4-20250514`, `gemini-2.0-flash`) |
| `language` | string | — | Language code (default: `en`) |
| `enabledTools` | string[] | — | Tools the agent can use (e.g., `["end_call", "transfer_call"]`) |
| `toolConfigs` | object | — | Per-tool configuration (transfer numbers, Cal.com keys, etc.) |
| `backgroundAudio` | object | — | Background audio preset |
| `backchannelEnabled` | boolean | — | Enable agent backchannel responses |
| `knowledgeFolderIds` | string[] | — | Knowledge base folder IDs to attach |
| `maxCallSeconds` | number | — | Max call duration (0 = unlimited) |
| `defaultDynamicVariables` | object | — | Default values for `{{variable}}` placeholders in prompt |
| `callSettings` | object | — | Call-level settings (voice speed, volume, etc.) |
| `fallbackVoice` | object | — | Fallback voice config if primary fails |
| `fallbackLlmModel` | string | — | Fallback LLM model |
| `postCallDataExtraction` | array | — | Fields to extract from calls via AI |
| `postCallExtractionModel` | string | — | Model to use for post-call extraction |
| `webhookUrl` | string | — | Post-call webhook URL |
| `agentType` | string | — | `"inbound"` or `"outbound"` |
| `role` | string | — | `"reception"`, `"booking"`, `"sales"`, or `"followup"` |
| `mode` | string | — | `"pipeline"`, `"native_audio"`, or `"openai_realtime"` |
