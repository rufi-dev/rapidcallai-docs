---
title: Get Agent
api: GET https://api.rapidcall.ai/api/agents/{id}
description: "Get a specific agent by ID"
---

<ParamField path="id" type="string" required>
  The agent ID. Example: `agent_abc123`
</ParamField>

## Request

```bash
curl -H "Authorization: Bearer YOUR_TOKEN" \
  https://api.rapidcall.ai/api/agents/agent_abc123
```
