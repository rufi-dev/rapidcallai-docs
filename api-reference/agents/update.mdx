---
title: Update Agent
api: PUT https://api.rapidcall.ai/api/agents/{id}
description: "Update an existing agent's configuration"
---

<ParamField path="id" type="string" required>
  The agent ID. Example: `agent_abc123`
</ParamField>

<ParamField body="name" type="string">
  Agent display name.
</ParamField>

<ParamField body="promptDraft" type="string">
  Draft system prompt.
</ParamField>

<ParamField body="promptPublished" type="string">
  Published (live) system prompt.
</ParamField>

<ParamField body="welcome" type="object">
  Welcome message configuration. See [Create Agent](/api-reference/agents/create) for nested fields.
</ParamField>

<ParamField body="voice" type="object">
  TTS voice configuration. See [Create Agent](/api-reference/agents/create) for nested fields.
</ParamField>

<ParamField body="llmModel" type="string">
  LLM model name (e.g., `"gpt-4o"`, `"gpt-4.1"`, `"claude-sonnet-4-20250514"`).
</ParamField>

<ParamField body="language" type="string">
  Language code.
</ParamField>

<ParamField body="enabledTools" type="string[]">
  Enabled tool list.
</ParamField>

<ParamField body="toolConfigs" type="object">
  Per-tool configuration.
</ParamField>

<ParamField body="webhookUrl" type="string">
  Post-call webhook URL.
</ParamField>

<ParamField body="agentType" type="string">
  `"inbound"` or `"outbound"`.
</ParamField>

<ParamField body="role" type="string">
  `"reception"`, `"booking"`, `"sales"`, or `"followup"`.
</ParamField>

All fields are optional -- only provided fields are updated.

## Request

```bash
curl -X PUT https://api.rapidcall.ai/api/agents/agent_abc123 \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Reception Bot v2",
    "promptDraft": "Updated instructions...",
    "llmModel": "gpt-4.1"
  }'
```

## Response

Returns the full updated agent object wrapped in `{ "agent": { ... } }`.

<Info>
Changes take effect on the next call. Active calls are not affected.
</Info>
