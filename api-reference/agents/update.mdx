---
title: Update Agent
api: PUT /api/agents/{id}
description: "Update an existing agent's configuration"
---

## Request

```bash
curl -X PUT https://api.rapidcall.ai/api/agents/agent_abc123 \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Reception Bot v2",
    "promptDraft": "Updated instructions...",
    "llmModel": "gpt-4.1"
  }'
```

All fields are optional — only provided fields are updated.

## Parameters

Same fields as [Create Agent](/api-reference/agents/create). All fields are optional.

| Field | Type | Description |
|-------|------|-------------|
| `name` | string | Agent display name |
| `promptDraft` | string | Draft system prompt |
| `promptPublished` | string | Published (live) system prompt |
| `welcome` | object | Welcome message configuration |
| `voice` | object | TTS voice configuration |
| `llmModel` | string | LLM model name |
| `language` | string | Language code |
| `enabledTools` | string[] | Enabled tool list |
| `toolConfigs` | object | Per-tool configuration |
| `webhookUrl` | string | Post-call webhook URL |
| `agentType` | string | `"inbound"` or `"outbound"` |
| `role` | string | `"reception"`, `"booking"`, `"sales"`, or `"followup"` |

## Response

Returns the full updated agent object wrapped in `{ "agent": { ... } }`.

<Info>
Changes take effect on the next call. Active calls are not affected.
</Info>
