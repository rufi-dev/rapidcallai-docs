---
title: List Calls
api: GET /api/calls
description: "Get call history for the current workspace"
---

<ParamField header="Authorization" type="string" required>
  Bearer token or API key. Example: `Bearer rck_your_api_key`
</ParamField>

<ParamField query="limit" type="number">
  Items per page (default: 20, max: 100).
</ParamField>

<ParamField query="offset" type="number">
  Number of items to skip (default: 0).
</ParamField>

<ParamField query="agent_id" type="string">
  Filter by agent ID.
</ParamField>

<ParamField query="from" type="string">
  Start date filter (ISO 8601 format).
</ParamField>

<ParamField query="to" type="string">
  End date filter (ISO 8601 format).
</ParamField>

## Request

```bash
curl -H "Authorization: Bearer YOUR_TOKEN" \
  "https://api.rapidcall.ai/api/calls?limit=20&offset=0"
```

## Response

```json
{
  "calls": [
    {
      "id": "call_abc123",
      "workspaceId": "ws_xyz",
      "agentId": "agent_xyz",
      "agentName": "Reception Bot",
      "to": "+15559876543",
      "fromNumber": "+15551234567",
      "direction": "inbound",
      "roomName": "room_abc123",
      "startedAt": 1705312200000,
      "endedAt": 1705312345000,
      "durationSec": 145,
      "outcome": "completed",
      "costUsd": 0.23,
      "transcript": [],
      "recording": null,
      "metrics": null,
      "analysisStatus": null,
      "postCallExtractionResults": [],
      "createdAt": "2025-01-15T10:30:00Z",
      "updatedAt": "2025-01-15T10:32:25Z"
    }
  ]
}
```

<Info>
- The `to` field contains the dialed number, `"webtest"` for web calls, or `"demo"` for demo calls.
- The `direction` field is `"inbound"`, `"outbound"`, or `"web"`.
- `costUsd` is the total cost charged to the workspace for this call.
</Info>
