---
title: List Calls
api: GET /api/calls
description: "Get call history for the current workspace"
---

## Request

```bash
curl -H "Authorization: Bearer YOUR_TOKEN" \
  "https://api.rapidcall.ai/api/calls?limit=20&offset=0"
```

### Query Parameters

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `limit` | number | 20 | Items per page (max 100) |
| `offset` | number | 0 | Skip N items |
| `agent_id` | string | — | Filter by agent |
| `from` | string | — | Start date (ISO 8601) |
| `to` | string | — | End date (ISO 8601) |

## Response

```json
{
  "calls": [
    {
      "id": "call_abc123",
      "workspaceId": "ws_xyz",
      "agentId": "agent_xyz",
      "agentName": "Reception Bot",
      "to": "+15559876543",
      "fromNumber": "+15551234567",
      "direction": "inbound",
      "roomName": "room_abc123",
      "startedAt": 1705312200000,
      "endedAt": 1705312345000,
      "durationSec": 145,
      "outcome": "completed",
      "costUsd": 0.23,
      "transcript": [],
      "recording": null,
      "metrics": null,
      "analysisStatus": null,
      "postCallExtractionResults": [],
      "createdAt": "2025-01-15T10:30:00Z",
      "updatedAt": "2025-01-15T10:32:25Z"
    }
  ]
}
```

<Info>
- The `to` field contains the dialed number, `"webtest"` for web calls, or `"demo"` for demo calls.
- The `direction` field is derived: `"inbound"`, `"outbound"`, or `"web"`.
- `durationSec` is computed from `startedAt`/`endedAt` if not stored directly.
- `costUsd` is the total cost charged to the workspace for this call.
</Info>
