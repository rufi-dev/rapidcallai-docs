---
title: Get Call
api: GET https://api.rapidcall.ai/api/calls/{id}
description: "Get detailed information about a specific call"
---

<ParamField path="id" type="string" required>
  The call ID. Example: `call_abc123`
</ParamField>

## Request

```bash
curl -H "Authorization: Bearer YOUR_TOKEN" \
  https://api.rapidcall.ai/api/calls/call_abc123
```

## Response

```json
{
  "call": {
    "id": "call_abc123",
    "workspaceId": "ws_xyz",
    "agentId": "agent_xyz",
    "agentName": "Reception Bot",
    "to": "+15559876543",
    "roomName": "room_abc123",
    "startedAt": 1705312200000,
    "endedAt": 1705312345000,
    "durationSec": 145,
    "outcome": "completed",
    "costUsd": 0.23,
    "transcript": [
      { "role": "agent", "text": "Thank you for calling Acme Dental, how can I help?", "timestamp": 0.0 },
      { "role": "user", "text": "I'd like to book an appointment", "timestamp": 3.2 },
      { "role": "agent", "text": "Of course! Let me check our availability...", "timestamp": 5.1 }
    ],
    "recording": {
      "kind": "egress_s3",
      "status": "ready",
      "durationSec": 145
    },
    "metrics": {
      "normalized": {
        "source": "inbound",
        "direction": "inbound"
      }
    },
    "analysisStatus": "completed",
    "postCallExtractionResults": [],
    "createdAt": "2025-01-15T10:30:00Z",
    "updatedAt": "2025-01-15T10:32:25Z"
  }
}
```

<Info>
- `outcome` values include: `"completed"`, `"no_answer"`, `"voicemail"`, `"call_transfer"`, `"in_progress"`, etc.
- `recording` is `null` if no recording exists. When present, check `recording.status === "ready"` before fetching.
- `transcript` is an array of message objects with `role` (`"agent"` or `"user"`) and `text`.
</Info>
