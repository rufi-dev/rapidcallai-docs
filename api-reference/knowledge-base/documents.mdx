---
title: Documents
description: "Upload and manage knowledge base documents"
---

## List Documents in Folder

```bash
curl -H "Authorization: Bearer YOUR_TOKEN" \
  https://your-server.com/api/kb/folders/folder_abc123/docs
```

### Response

```json
[
  {
    "id": "doc_abc123",
    "title": "Business Hours",
    "type": "text",
    "char_count": 450,
    "created_at": "2025-01-10T09:00:00Z"
  },
  {
    "id": "doc_def456",
    "title": "Services Catalog.pdf",
    "type": "pdf",
    "char_count": 12500,
    "created_at": "2025-01-12T11:00:00Z"
  }
]
```

## Add Text Document

```bash
curl -X POST https://your-server.com/api/kb/folders/folder_abc123/text \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "title": "Business Hours",
    "content": "We are open Monday through Friday, 9 AM to 5 PM EST. Saturday 10 AM to 2 PM. Closed Sundays and holidays."
  }'
```

## Upload PDF

```bash
curl -X POST https://your-server.com/api/kb/folders/folder_abc123/pdf \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -F "file=@services-catalog.pdf"
```

The PDF is automatically parsed, chunked, and embedded for semantic search.

## Update Document

```bash
curl -X PUT https://your-server.com/api/kb/docs/doc_abc123 \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "title": "Updated Business Hours",
    "content": "New hours: Monday-Saturday 8 AM to 6 PM."
  }'
```

## Delete Document

```bash
curl -X DELETE https://your-server.com/api/kb/docs/doc_abc123 \
  -H "Authorization: Bearer YOUR_TOKEN"
```
