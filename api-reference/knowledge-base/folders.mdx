---
title: Knowledge Base Folders
description: "Manage knowledge base folders"
---

## List Folders

```bash
curl -H "Authorization: Bearer YOUR_TOKEN" \
  https://your-server.com/api/kb/folders
```

### Response

```json
[
  {
    "id": "folder_abc123",
    "name": "Product FAQ",
    "doc_count": 12,
    "created_at": "2025-01-10T09:00:00Z"
  }
]
```

## Create Folder

```bash
curl -X POST https://your-server.com/api/kb/folders \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"name": "Product FAQ"}'
```

### Response

```json
{
  "id": "folder_new456",
  "name": "Product FAQ",
  "doc_count": 0,
  "created_at": "2025-01-15T14:00:00Z"
}
```

## Update Folder

```bash
curl -X PUT https://your-server.com/api/kb/folders/folder_abc123 \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"name": "Updated FAQ"}'
```

## Delete Folder

```bash
curl -X DELETE https://your-server.com/api/kb/folders/folder_abc123 \
  -H "Authorization: Bearer YOUR_TOKEN"
```

<Warning>
Deleting a folder removes all documents and embeddings inside it. Agents that reference this folder will no longer have access to its content.
</Warning>
