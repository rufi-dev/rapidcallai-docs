---
title: "Export Call Data"
api: GET https://api.rapidcall.ai/api/calls/{id}/export
description: "Export complete call data including transcript, metrics, and recording URL."
---

Returns a comprehensive export of all data associated with a call. Includes the full transcript, call metrics, cost breakdown, recording URL, evaluations, and labels. Useful for compliance archiving or integration with external analytics tools.

<ParamField path="id" type="string" required>
  The unique identifier of the call to export.
</ParamField>

<RequestExample>
```bash cURL
curl -X GET https://api.rapidcall.ai/api/calls/call_8f3a2b1c4d5e/export \
  -H "Authorization: Bearer YOUR_API_KEY"
```
</RequestExample>
