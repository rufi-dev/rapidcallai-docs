---
title: "List Evaluations"
api: GET https://api.rapidcall.ai/api/calls/{id}/evaluations
description: "Retrieve all evaluations (manual and AI-generated) for a specific call."
---

Returns a list of evaluations attached to a call, including both manual evaluations added by team members and AI auto-evaluations.

<ParamField path="id" type="string" required>
  The unique identifier of the call.
</ParamField>

<RequestExample>
```bash cURL
curl -X GET https://api.rapidcall.ai/api/calls/call_8f3a2b1c4d5e/evaluations \
  -H "Authorization: Bearer YOUR_API_KEY"
```
</RequestExample>
