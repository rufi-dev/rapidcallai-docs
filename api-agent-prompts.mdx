---
title: "AI Prompt Tools API"
description: "Use AI to generate and refine agent prompts."
---

These endpoints use AI to help you write high-quality system prompts for your voice agents. Provide a brief description of what your agent should do, and the AI generates a complete, production-ready prompt.

## Generate prompt

```http
POST /api/agents/generate-prompt
Content-Type: application/json
Authorization: Bearer YOUR_TOKEN
```

Generate a full system prompt from a brief natural-language description of what the agent should do.

**Body:**
```json
{
  "description": "A friendly dental office receptionist that books appointments, answers questions about services and insurance, and transfers to staff when needed."
}
```

**curl:**
```bash
curl -X POST https://api.rapidcall.ai/api/agents/generate-prompt \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "description": "A friendly dental office receptionist that books appointments, answers questions about services and insurance, and transfers to staff when needed."
  }'
```

**Parameters:**

| Parameter     | Type   | Required | Description                                                          |
|---------------|--------|----------|----------------------------------------------------------------------|
| `description` | string | Yes      | Brief description of the agent's role, personality, and tasks.       |

**Response `200`:**
```json
{
  "prompt": "You are a warm and professional receptionist for a dental office. Your primary responsibilities are:\n\n1. **Appointment Booking** -- Help callers schedule, reschedule, or cancel appointments...\n\n2. **Service Information** -- Answer questions about dental services, pricing estimates, and accepted insurance plans...\n\n3. **Transfers** -- If the caller needs to speak with a dentist or office manager, use the transfer_call tool...\n\n## Tone & Style\n- Friendly, patient, and reassuring\n- Use simple language, avoid medical jargon\n- Confirm details by reading them back to the caller\n..."
}
```

## Prompt assist

```http
POST /api/agents/prompt-assist
Content-Type: application/json
Authorization: Bearer YOUR_TOKEN
```

Refine or improve an existing prompt. Provide the current prompt along with instructions for how it should be changed.

**Body:**
```json
{
  "currentPrompt": "You are a customer support agent. Help users with their orders.",
  "instructions": "Make it more detailed, add handling for refund requests, and make the tone more empathetic."
}
```

**curl:**
```bash
curl -X POST https://api.rapidcall.ai/api/agents/prompt-assist \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "currentPrompt": "You are a customer support agent. Help users with their orders.",
    "instructions": "Make it more detailed, add handling for refund requests, and make the tone more empathetic."
  }'
```

**Parameters:**

| Parameter       | Type   | Required | Description                                                          |
|-----------------|--------|----------|----------------------------------------------------------------------|
| `currentPrompt` | string | Yes      | The existing prompt you want to improve.                             |
| `instructions`  | string | Yes      | What changes or improvements you want the AI to make.                |

**Response `200`:**
```json
{
  "prompt": "You are a compassionate and knowledgeable customer support agent. Your goal is to make every caller feel heard and supported.\n\n## Order Assistance\n- Look up order status using the caller's name or order number\n- Provide shipping updates and estimated delivery dates\n- Help with order modifications if the order has not yet shipped\n\n## Refund Requests\n- Listen to the customer's concern with empathy before offering solutions\n- If the item is within the 30-day return window, initiate the refund process\n- For items outside the return window, offer store credit or escalate to a manager\n..."
}
```
