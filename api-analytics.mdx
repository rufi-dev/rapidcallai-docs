## Workspace analytics

```http
GET /api/analytics
Authorization: Bearer YOUR_TOKEN
```

Returns aggregate analytics for the entire workspace over a given time period.

**Query parameters:**

| Parameter | Type   | Required | Description                                                   |
|-----------|--------|----------|---------------------------------------------------------------|
| `from`    | string | No       | Start date in ISO 8601 format. Defaults to 30 days ago.      |
| `to`      | string | No       | End date in ISO 8601 format. Defaults to now.                 |

**curl:**
```bash
curl "https://api.rapidcall.ai/api/analytics?from=2026-03-01T00:00:00Z&to=2026-03-31T23:59:59Z" \
  -H "Authorization: Bearer YOUR_TOKEN"
```

**Response `200`:**
```json
{
  "totalCalls": 1284,
  "totalDurationMinutes": 642.5,
  "averageDurationSeconds": 30.0,
  "averageLatencyMs": 820,
  "outcomes": {
    "completed": 1050,
    "voicemail": 98,
    "call_transfer": 72,
    "no_answer": 41,
    "failed": 23
  },
  "callsByDay": [
    { "date": "2026-03-01", "count": 42 },
    { "date": "2026-03-02", "count": 38 }
  ],
  "sentimentBreakdown": {
    "positive": 680,
    "neutral": 412,
    "negative": 192
  }
}
```

Authorization: Bearer YOUR_TOKEN

## Create dashboard

```http
POST /api/dashboards
Content-Type: application/json
Authorization: Bearer YOUR_TOKEN
```

Create a new custom dashboard.

**Body:**
```json
{
  "name": "Support Metrics"
}
```

**curl:**
```bash
curl -X POST https://api.rapidcall.ai/api/dashboards \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"name": "Support Metrics"}'
```

**Parameters:**

| Parameter | Type   | Required | Description              |
|-----------|--------|----------|--------------------------|
| `name`    | string | Yes      | Name of the dashboard.   |

**Response `201`:**
```json
{
  "dashboard": {
    "id": "dash_002",
    "name": "Support Metrics",
    "widgets": [],
    "createdAt": "2026-04-01T12:00:00Z"
  }
}
```

## Add widget to dashboard

```http
POST /api/dashboards/:id/widgets
Content-Type: application/json
Authorization: Bearer YOUR_TOKEN
```

Add a widget (chart, counter, table) to a dashboard.

**Body:**
```json
{
  "type": "line_chart",
  "title": "Daily Call Volume",
  "config": {
    "metric": "call_count",
    "groupBy": "day",
    "agentId": "g4d3cXIpDr"
  }
}
```

**curl:**
```bash
curl -X POST https://api.rapidcall.ai/api/dashboards/dash_002/widgets \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "type": "line_chart",
    "title": "Daily Call Volume",
    "config": {
      "metric": "call_count",
      "groupBy": "day",
      "agentId": "g4d3cXIpDr"
    }
  }'
```

**Parameters:**

| Parameter | Type   | Required | Description                                                             |
|-----------|--------|----------|-------------------------------------------------------------------------|
| `type`    | string | Yes      | Widget type: `line_chart`, `bar_chart`, `counter`, `table`, `pie_chart`.|
| `title`   | string | Yes      | Display title for the widget.                                           |
| `config`  | object | Yes      | Widget-specific configuration (metric, filters, grouping).              |

**Response `201`:**
```json
{
  "widget": {
    "id": "wgt_001",
    "type": "line_chart",
    "title": "Daily Call Volume",
    "config": {
      "metric": "call_count",
      "groupBy": "day",
      "agentId": "g4d3cXIpDr"
    }
  }
}
```

## Get widget data

```http
POST /api/dashboards/:dashId/widgets/:widgetId/data
Content-Type: application/json
Authorization: Bearer YOUR_TOKEN
```

Fetch the rendered data for a specific widget, optionally overriding the time range.

**Body:**
```json
{
  "from": "2026-03-01T00:00:00Z",
  "to": "2026-03-31T23:59:59Z"
}
```

**curl:**
```bash
curl -X POST https://api.rapidcall.ai/api/dashboards/dash_002/widgets/wgt_001/data \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "from": "2026-03-01T00:00:00Z",
    "to": "2026-03-31T23:59:59Z"
  }'
```

**Parameters:**

| Parameter | Type   | Required | Description                                         |
|-----------|--------|----------|-----------------------------------------------------|
| `from`    | string | No       | Start date in ISO 8601 format.                      |
| `to`      | string | No       | End date in ISO 8601 format.                        |

**Response `200`:**
```json
{
  "widgetId": "wgt_001",
  "type": "line_chart",
  "data": {
    "labels": ["2026-03-01", "2026-03-02", "2026-03-03"],
    "values": [42, 38, 55]
  }
}
```
