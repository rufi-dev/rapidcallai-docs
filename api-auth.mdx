## Register

```http
POST /api/auth/register
Content-Type: application/json
```

Create a new account and workspace.

**Body:**
```json
{
  "email": "you@company.com",
  "password": "securePassword123",
  "name": "Jane Smith"
}
```

**curl:**
```bash
curl -X POST https://api.rapidcall.ai/api/auth/register \
  -H "Content-Type: application/json" \
  -d '{
    "email": "you@company.com",
    "password": "securePassword123",
    "name": "Jane Smith"
  }'
```

**Parameters:**

| Parameter  | Type   | Required | Description                        |
|------------|--------|----------|------------------------------------|
| `email`    | string | Yes      | Valid email address.               |
| `password` | string | Yes      | Minimum 8 characters.              |
| `name`     | string | Yes      | Full name of the account owner.    |

**Response `201`:**
```json
{
  "token": "eyJhbGciOiJIUzI1NiIs...",
  "user": {
    "id": "u_abc123",
    "email": "you@company.com",
    "name": "Jane Smith"
  },
  "workspace": {
    "id": "ws_def456",
    "name": "Jane Smith's Workspace"
  }
}
```

## Google OAuth

```http
POST /api/auth/google
Content-Type: application/json
```

Authenticate or register using a Google OAuth token.

**Body:**
```json
{
  "token": "google-oauth-id-token"
}
```

**curl:**
```bash
curl -X POST https://api.rapidcall.ai/api/auth/google \
  -H "Content-Type: application/json" \
  -d '{"token": "google-oauth-id-token"}'
```

**Parameters:**

| Parameter | Type   | Required | Description                          |
|-----------|--------|----------|--------------------------------------|
| `token`   | string | Yes      | Google OAuth ID token from client.   |

**Response `200`:**
```json
{
  "token": "eyJhbGciOiJIUzI1NiIs...",
  "user": {
    "id": "u_abc123",
    "email": "you@gmail.com",
    "name": "Jane Smith"
  },
  "workspace": {
    "id": "ws_def456",
    "name": "Jane Smith's Workspace"
  }
}
```

## Log out

```http
POST /api/auth/logout
Authorization: Bearer YOUR_TOKEN
```

Invalidate the current session token.

**curl:**
```bash
curl -X POST https://api.rapidcall.ai/api/auth/logout \
  -H "Authorization: Bearer YOUR_TOKEN"
```

**Response `200`:**
```json
{
  "message": "Logged out successfully"
}
```

## Reset password

```http
POST /api/auth/reset-password
Content-Type: application/json
```

Set a new password using the token from the reset email.

**Body:**
```json
{
  "token": "reset-token-from-email",
  "password": "newSecurePassword456"
}
```

**curl:**
```bash
curl -X POST https://api.rapidcall.ai/api/auth/reset-password \
  -H "Content-Type: application/json" \
  -d '{
    "token": "reset-token-from-email",
    "password": "newSecurePassword456"
  }'
```

**Parameters:**

| Parameter  | Type   | Required | Description                       |
|------------|--------|----------|-----------------------------------|
| `token`    | string | Yes      | Reset token from the email link.  |
| `password` | string | Yes      | New password (minimum 8 chars).   |

**Response `200`:**
```json
{
  "message": "Password has been reset successfully."
}
```

## Get current user

```http
GET /api/me
Authorization: Bearer YOUR_TOKEN
```

Returns the authenticated user's profile and active workspace.

**curl:**
```bash
curl https://api.rapidcall.ai/api/me \
  -H "Authorization: Bearer YOUR_TOKEN"
```

**Response `200`:**
```json
{
  "user": {
    "id": "u_abc123",
    "email": "you@company.com",
    "name": "Jane Smith",
    "emailVerified": true,
    "createdAt": "2026-03-01T12:00:00Z"
  },
  "workspace": {
    "id": "ws_def456",
    "name": "Jane Smith's Workspace",
    "hasPaymentMethod": true,
    "suspended": false
  }
}
```
