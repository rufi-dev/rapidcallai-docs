# Welcome to RapidCall AI

RapidCall AI is a platform for building AI-powered voice agents. Create intelligent phone agents that answer calls, qualify leads, book appointments, provide customer support, and transfer calls -- all with natural-sounding voices and real-time conversation.

## Key Features

- **Voice Agents** -- Configure personality, voice, language, LLM provider, tools, and knowledge base per agent
- **Workflow Builder** -- Visual drag-and-drop canvas with multiple node types and pre-built templates
- **Telephony** -- Inbound/outbound calls with call transfer support
- **Knowledge Base** -- Upload PDFs and text documents for RAG-powered conversations
- **Outbound Campaigns** -- Batch calling with CSV upload, scheduling, and DNC management
- **Analytics** -- Call volume, duration, cost, latency metrics with custom dashboards
- **Webhooks** -- Real-time notifications when calls end with full transcript and metadata
- **A/B Testing** -- Test different prompts, voices, and models across agent variants
- **API** -- Full REST API for programmatic access to all platform features
