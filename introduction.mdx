# Welcome to RapidCall AI

RapidCall AI is an open-source platform for building AI-powered voice agents. Create intelligent phone agents that can answer calls, qualify leads, book appointments, provide customer support, and transfer calls — all with natural-sounding voices and real-time conversation.

## Platform Overview

RapidCall AI consists of four main components:

| Component | Technology | Description |
|-----------|-----------|-------------|
| **Server** | Node.js / Express | REST API, auth, billing, telephony orchestration |
| **Voice Agent** | Python / LiveKit | Real-time voice processing, STT/TTS, LLM integration |
| **Client** | React / Vite | Dashboard UI with workflow builder, analytics, CRM |
| **Database** | PostgreSQL | Multi-tenant data storage with workspace isolation |

## Key Features

- **Voice Agents** — Configure personality, voice, language, LLM provider, tools, and knowledge base per agent
- **Workflow Builder** — Visual drag-and-drop canvas with 9 node types and pre-built templates
- **Telephony** — Twilio integration for inbound/outbound calls with call transfer support
- **Knowledge Base** — Upload PDFs and text documents for RAG-powered conversations
- **CRM** — Contact management with call history, tags, and notes
- **Outbound Campaigns** — Batch calling with CSV upload, scheduling, and DNC management
- **Analytics** — Call volume, duration, cost, latency metrics with custom dashboards
- **Billing** — Usage-based billing with Lago/Stripe integration
- **API Keys** — Programmatic access to all platform features
