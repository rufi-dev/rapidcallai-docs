## List evaluations

```http
GET /api/calls/:id/evaluations
Authorization: Bearer YOUR_TOKEN
```

Returns all evaluations (manual and AI-generated) for a call.

**curl:**
```bash
curl https://api.rapidcall.ai/api/calls/call_abc123/evaluations \
  -H "Authorization: Bearer YOUR_TOKEN"
```

**Response `200`:**
```json
{
  "evaluations": [
    {
      "id": "eval_001",
      "type": "manual",
      "score": 4,
      "notes": "Agent handled the refund request well but was slow to respond.",
      "createdBy": "u_abc123",
      "createdAt": "2026-03-25T14:00:00Z"
    },
    {
      "id": "eval_002",
      "type": "auto",
      "score": 3,
      "notes": "AI evaluation: Agent followed the script but missed an upsell opportunity.",
      "createdAt": "2026-03-25T14:05:00Z"
    }
  ]
}
```

## Add manual evaluation

```http
POST /api/calls/:id/evaluations
Content-Type: application/json
Authorization: Bearer YOUR_TOKEN
```

Add a manual evaluation to a call.

**Body:**
```json
{
  "score": 5,
  "notes": "Excellent call. Agent was friendly and resolved the issue quickly."
}
```

**curl:**
```bash
curl -X POST https://api.rapidcall.ai/api/calls/call_abc123/evaluations \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "score": 5,
    "notes": "Excellent call. Agent was friendly and resolved the issue quickly."
  }'
```

**Parameters:**

| Parameter | Type   | Required | Description                                |
|-----------|--------|----------|--------------------------------------------|
| `score`   | number | Yes      | Rating from 1 (poor) to 5 (excellent).    |
| `notes`   | string | No       | Free-text notes about the call quality.    |

**Response `201`:**
```json
{
  "evaluation": {
    "id": "eval_003",
    "type": "manual",
    "score": 5,
    "notes": "Excellent call. Agent was friendly and resolved the issue quickly.",
    "createdBy": "u_abc123",
    "createdAt": "2026-04-01T12:00:00Z"
  }
}
```

## Trigger AI auto-evaluation

```http
POST /api/calls/:id/auto-evaluate
Authorization: Bearer YOUR_TOKEN
```

Trigger an AI-powered evaluation of the call. The AI reviews the transcript and assigns a score with detailed feedback.

**curl:**
```bash
curl -X POST https://api.rapidcall.ai/api/calls/call_abc123/auto-evaluate \
  -H "Authorization: Bearer YOUR_TOKEN"
```

**Response `200`:**
```json
{
  "evaluation": {
    "id": "eval_004",
    "type": "auto",
    "score": 4,
    "notes": "The agent maintained a professional tone throughout the call. Greeting was strong. The agent correctly identified the customer's issue and provided an accurate resolution. Minor improvement: the agent could have proactively offered a follow-up call.",
    "criteria": {
      "greeting": 5,
      "problemIdentification": 4,
      "resolution": 4,
      "tone": 5,
      "closing": 3
    },
    "createdAt": "2026-04-01T12:05:00Z"
  }
}
```

## Add label

```http
POST /api/calls/:id/labels
Content-Type: application/json
Authorization: Bearer YOUR_TOKEN
```

Apply a label to a call.

**Body:**
```json
{
  "name": "escalated",
  "color": "#8b5cf6"
}
```

**curl:**
```bash
curl -X POST https://api.rapidcall.ai/api/calls/call_abc123/labels \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"name": "escalated", "color": "#8b5cf6"}'
```

**Parameters:**

| Parameter | Type   | Required | Description                                      |
|-----------|--------|----------|--------------------------------------------------|
| `name`    | string | Yes      | Label name. Reuses existing label if name matches.|
| `color`   | string | No       | Hex color code for the label badge.              |

**Response `201`:**
```json
{
  "label": {
    "id": "lbl_003",
    "name": "escalated",
    "color": "#8b5cf6"
  }
}
```

## Remove label

```http
DELETE /api/calls/:id/labels
Content-Type: application/json
Authorization: Bearer YOUR_TOKEN
```

Remove a label from a call.

**Body:**
```json
{
  "name": "needs-review"
}
```

**curl:**
```bash
curl -X DELETE https://api.rapidcall.ai/api/calls/call_abc123/labels \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"name": "needs-review"}'
```

**Parameters:**

| Parameter | Type   | Required | Description                |
|-----------|--------|----------|----------------------------|
| `name`    | string | Yes      | Name of the label to remove.|

**Response `200`:**
```json
{
  "message": "Label removed."
}
```

## Export call data

```http
GET /api/calls/:id/export
Authorization: Bearer YOUR_TOKEN
```

Export all data for a call, including transcript, evaluations, labels, recording URL, and analysis.

**curl:**
```bash
curl https://api.rapidcall.ai/api/calls/call_abc123/export \
  -H "Authorization: Bearer YOUR_TOKEN"
```

**Response `200`:**
```json
{
  "call": {
    "id": "call_abc123",
    "agentId": "g4d3cXIpDr",
    "agentName": "Sales Agent",
    "from": "+14155550100",
    "to": "+14155550200",
    "direction": "inbound",
    "outcome": "completed",
    "durationSeconds": 145,
    "startedAt": "2026-03-25T13:30:00Z",
    "endedAt": "2026-03-25T13:32:25Z"
  },
  "transcript": [
    { "role": "agent", "content": "Hello! Thanks for calling Acme Corp. How can I help?", "timestamp": 0.0 },
    { "role": "user", "content": "I need to check on my order status.", "timestamp": 3.2 }
  ],
  "evaluations": [
    { "type": "manual", "score": 5, "notes": "Great call." }
  ],
  "labels": ["high-priority"],
  "analysis": {
    "summary": "Customer called to check order status. Agent provided tracking information.",
    "sentiment": "positive"
  },
  "recordingUrl": "https://api.rapidcall.ai/api/calls/call_abc123/recording"
}
```
